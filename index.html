Mode 1

In this mode, the robot can draw out figures and letters based on online user inputs. There is an online sketchpad that the user can draw on with a mouse or electronic pencil such as the Apple pencil. The user can draw shapes and write text onto the online sketchpad. After finalizing the desired drawings and texts, press the button "Draw" to let the robot to start drawing. Currently, this mode is restricted to drawing with one color and one drawing tool (brush, crayon, pencil crayon, etc). If multiple colors and drawing tools are going to be needed, refer to mode 2 or divide the drawing into smaller pieces accordingly.

Start Drawing

Mode 2

In this mode, the robot actions can be controlled with an external controller. There are four buttons that controls the wheel movement, and one button that controls the height which the robot pen/pencil will be held at. By doing so, the robot can be controlled externally to draw out shapes or text based on the user's interest If multiple colors and drawing tools (brush, crayon, pencil crayon, etc) are doing to be used, the tools will have to be manually switched during the drawing process.
