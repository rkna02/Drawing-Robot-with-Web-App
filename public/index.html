Mode 1

In this mode, the robot can draw out figures and letters based on online user inputs.
User can either choose:
Option 1: click a shape to draw, or
Option 2: enter text or a command and press "Submit" to make the robot write texts.

Currently, this mode is restricted to drawing with one color and one drawing tool (brush, crayon, pencil crayon, etc). If multiple colors and drawing tools are going to be needed, refer to mode 2 or divide the drawing into smaller pieces accordingly.

Start Drawing

Mode 2

In this mode, the robot actions can be controlled with user defined directions.
"↖↑↗ ↙↓↘" controls the wheel movement,
"⇈ ⇊" controls the pen lifting up and down.
By doing so, the robot can be controlled externally to move around and draw out shapes or texts based on the user's interest.

If multiple colors and drawing tools (brush, crayon, pencil crayon, etc) are going to be used, the tools will have to be manually switched during the drawing process.
